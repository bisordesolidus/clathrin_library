# Спектральная модель клатриновой оболочки

Коарс-грейн упругая (ENM/ANM-подобная) SE(3)-модель клатриновой решётки:
нормальные моды, теория возмущений по мембранному натяжению (Helfrich),
критическое натяжение, зонная структура объёмов. Клетки — фуллереновые
полиэдры; отбор допустимых — правило запрета «голова-хвост» (Schein 2009).
Геометрия трискелиона измерена из PDB 6SCT.

```bash
pip install -r requirements.txt
```

Python 3.10+. Зависимости: numpy, scipy, networkx, matplotlib.


Весь пайплайн — через `main.py`:

```bash
python main.py                    # все клетки Шайна n<=50: для каждой рисунки
                                  # (cage, spectrum, soft_mode, buckling_mode)
                                  # + отчёт + общая зонная диаграмма объёмов
python main.py --cage barrel-36   # одна клетка (dodeca-20 | barrel-36 |
                                  #   minicoat-28 | C<n>-<k>)
python main.py --n 20 24 28 32    # выбранные C_n (все разрешённые Шайном)
python main.py --n 60             # включить C60 (ДОЛГО, ~2ч, 226M кандидатов;
                                  #   для кластера)
python main.py --bands-only       # только зонная диаграмма из кэша
```

Выход — в `runs/`: по папке на клетку (4 PNG + report.md) и общая
`volume_bands.png` + `bands_report.md`. Дуалы клеток кэшируются в `runs/cache/`
(наращиваются по n, не пересчитываются).

Ядро без графики (чистое, тестируемое):

| Модуль | Назначение |
|---|---|
| `se3.py` | Алгебра SE(3): exp/log/Ad/ad |
| `triskelion.py` | Геометрия трискелиона (ψ, χ из 6SCT) |
| `lattice.py`, `embed.py` | Контейнер решётки, вложение фуллерена |
| `fullerene.py` | Спиральный генератор клеток |
| `enumerate_par.py` | Параллельное перечисление (большие n) |
| `routing.py` | Укладка ног, правило «голова-хвост» |
| `schein.py` | Классификатор запрета Шайна |
| `hamiltonian.py`, `diagonalize.py` | Гессиан H₀, спектр, классификация мод |
| `rigidbody.py`, `relax.py` | SE(3)-модель, релаксация фрустрации |
| `perturbation.py` | Теория возмущений (натяжение) |
| `volume.py`, `bandstructure.py` | Площадь/объём, восприимчивости, зоны |
| `stability.py` | Критическое натяжение (buckling) |
| `visualize.py`, `main.py` | Рисунки и оркестрация |

Физическая модель и формулы — в `MODEL.md`.
